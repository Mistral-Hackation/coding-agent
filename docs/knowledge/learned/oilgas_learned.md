## Thread Specifications (2026-01-26 17:38)

*Source: Review Finding - Threaded Fitting Adapter Analysis*

ISO Metric Thread Geometry Standards:
- Thread depth (external) = 0.6134 × pitch
- Thread depth (internal) = 0.5413 × pitch
- Thread angle = 60° symmetric
- For 1/2" fittings: NPT/BSPT use 14 TPI (1.814mm pitch), thread depth ~0.87mm
- M30 metric: coarse pitch = 3.5mm, fine pitch = 2.0mm
- Common error: Using arbitrary pitch/depth ratios instead of standard formulas
- For visualization, ±10% deviation acceptable; for manufacturing CAD, must be exact
